# Онлайн-запис на автомийку

Веб-додаток Flask для бронювання послуг автомийки.

## Запуск

1. Встановити залежності:
   ```
   pip install -r requirements.txt
   ```
2. Скопіювати `.env.example` у `.env` та змінити параметри за потреби.
3. Запустити сервер:
   ```
   python app.py
   ```

База даних зберігається у файлі `carwash.db` (SQLite).
